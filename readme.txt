== Description ==

Validates that the email entered by the user matches an email within a list of emails during the registration process. In the registration process, if the email entered by the user does not match any from the list, an error message is displayed. This message is customizable.

== Installation ==

1. Upload the `email-validation-list` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Email Validation List' section under 'Settings' in the WordPress admin area.
4. Enter a list of allowed emails and customize the error message.
5. Save the settings.

== Frequently Asked Questions ==

= How do I set up the allowed emails? =

In the WordPress admin area, go to 'Settings' -> 'Email Validation List'. There, you can enter a list of allowed emails separated by new lines.

= Can I customize the error message? =

Yes, you can customize the error message that is displayed when an invalid email is entered during the registration process.
